## **Overview of the Website Structure**

The website consists of five main pages, each representing cars from a specific country:

- German Cars

- Japanese Cars

- British Cars

- Italian Cars

- American Cars

Each page has the same structure:

- Header & navigation menu

- History and Intro section

- Brand-focused sections (BMW, Audi, Mercedes, etc.) with images and details

- Footer

- All pages are linked via the navigation menu at the top of each page.

## **Instructions for Updating Text Content**

1. Open the HTML file for the page you want to update (e.g., german.html).

2. Locate the paragraphs or links elements you want to edit.

3. Update the text inside the tags.

4. Save the file and refresh your browser to view new changes

## **Instructions for Adding New Pages**

1. Create a new HTML file (e.g., french.html).

2. Copy the structure from an existing page (e.g., german.html).

3. Update:

(title) tag

(h1) page heading

Brand sections, text, and images

4. Add a link to the new page in the navigation menu of all pages

## **Instructions for Adding or Replacing Images**

1. Place the new image in the assests/ folder.

2. Replace the src in the <img> tag.

3. Ensure the alt attribute describes the image for accessibility

## **Instructions for Modifying Navigation**

1. Open the <nav> section in each HTML page.

2. Add, remove, or edit <li> links as needed.

3. Save all pages after updating to ensure the menu stays consistent.

## **Instructions for Uploading Changes**

1. Connect to your hosting platform (e.g GitHub Pages).

2. Replace the existing HTML, CSS, and image files with the updated files.

3. Refresh the site in a browser to confirm that changes are live.

## **Common Troubleshooting Scenarios**

Problem--> Solution

1. Page doesn’t update -->	Clear your browser cache and refresh the page

2. Images don’t show -->Check the image path and filename spelling

3. Navigation links broken -->	Ensure all <a href=""> paths are correct and match filenames

4. Layout looks broken --->	Check CSS file link in <head> is correct (<link rel="stylesheet" href="style.css">)

## **Recommended Maintenance Schedule**

- Weekly: Check that all links work and images display correctly.

- Monthly: Review and update content (text, images, popular models).

- After Major Updates: Test navigation, responsiveness, and layout on multiple devices.

- Yearly: Consider updating color scheme, typography, or layout if trends changes.
